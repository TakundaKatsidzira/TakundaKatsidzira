# Sequence Game

This project implements the classic board game *Sequence* playable via CLI, supporting Human vs Computer mode.  
Features include AI opponent with heuristic and minimax, card/board management, and detailed JSON logging.

## Installation

Run the setup script to prepare the environment:

```bash
./setup.sh
```

## Usage

Start the game with:

```bash
python3 src/sequence.py
```

## Project Structure

- `src/` - Source code modules  
- `tests/` - Unit tests  
- `data/` - Logs and saved data  
- `docs/` - Documentation and gameplay rules
